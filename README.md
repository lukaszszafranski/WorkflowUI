## About The Project
This project is frontend part of a bigger project: Kanban board with real-time chat. It was built with Angular 9.

### Built With
* Angular 9 
* SignalR
* Font Awesome
* ngx-spinner
* Tawk.to
* ngx-toastr
* rxjs 
* core-js
* Angular Material Design (CDK)
* Azure App Service
* Compodoc
